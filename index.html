About this project

This was one of my first projects. It is simple and doesn't have much functionality, but it helped me get better at using forms in HTML/CSS. I also learned how to style and validate them in my own way, overriding the browser's default functionality. The special highlight of this project was working with design and theme; I generated images using neural networks and adapted them to fit the project. The theme of the project and the beaver motif are a reference to my friend Bobur, who also works in web design and whose name sounds similar to a famous Polish meme about a beaver.

P.S. If you click on the project, it will open in a new window and won't interfere with the game. However, you don't need to worry about this and can continue playing, as links to all projects will be provided again at the end of the game.
